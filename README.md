# 🎤 AI Interview Coach

An AI-powered interview preparation platform that simulates real technical and HR interviews using voice, speech analysis, and Generative AI. It provides real-time feedback on communication, confidence, eye contact, and interview performance.

## 🚀 Features

- 🤖 AI-generated interview questions
- 🎙️ Voice-based interview simulation
- 📝 Speech-to-Text using OpenAI Whisper
- 👀 Eye contact detection with MediaPipe
- 😊 Facial emotion analysis
- 💬 AI-generated feedback and suggestions
- 📊 Confidence and communication score
- 📄 Downloadable interview report
- 📚 Technical and HR interview modes
- 🔒 Secure authentication

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/rat224/ai-interview-coach.git
cd ai-interview-coach
```

### Create Virtual Environment

```bash
python -m venv .venv
```

### Activate Environment

Windows

```bash
.venv\Scripts\activate
```

Linux / macOS

```bash
source .venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Backend

```bash
uvicorn app:app --reload
```
